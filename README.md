## Installation

```bash
$ cd sumatywny-api
$ npm install
```

## Running the app

### Start database Docker images
```bash
$ sudo docker-compose up -d
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
